# Hebrew Word Tools
A professional VBA Add-in for Microsoft Word to simplify Hebrew document editing.

### Features
* **Key Fix:** Instantly switch text typed in the wrong keyboard layout (English <-> Hebrew).
* **Atbash:** Perform Atbash ciphers while preserving vowels (Nekudos) and fixing Final Letters (Sofiyot).

### How to Install
1. Download `Install_HebrewTools.bat`.
2. Run the file.
3. Restart Microsoft Word.

*Note: You may need to click "More Info" -> "Run Anyway" if Windows shows a security warning.*
